

DB Manager
	Assist in database management as well as data analysis by working with web based data collection, Access and other relational databases and by synthesizing and analyzing qualitative/quantitative data, producing statistical and written reports, summaries and projections and reporting data to external groups.

	Provide computer-based administrative support in the following areas – development and maintenance of data and information inventory systems, as well as committee support as required.

	Duties and Responsibilities:
	· Manage web based database and other relational databases as appropriate
	· Co-ordinate relational databases as needed and design systems to accommodate changes in subsequent years
	· Determine the availability of data from existing sources
	· Cross reference various sources of data to generate reports and ensure output is accessible and user friendly
	· Synthesize and analyze qualitative/quantitative data
	· Also responsible for special projects
